Testing ruby product 'keydown'

May 4, 2011

I tried to install keydown in my MacOSX and I wrote it down here.

git clone

At first, mkdir project directory and chage directory

        % mkdir -p ~/project
        % cd ~/project

Next, implement git command 'git clone'

        % git clone https://github.com/infews/keydown.git

you've got keydown source code

PATH & RUBYLIB

% echo 'RUBYLIB=$HOME/project/keydown/lib:$RUBYLIB
export RUBYLIB
 
PATH=$HOME/project/keydown/bin:$PATH
export PATH
' >> ~/.zshrc_custom
% source ~/.zshrc_custom
